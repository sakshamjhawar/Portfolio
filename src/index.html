Here's something about this project

NumberNinja

Web application to aid Grade 2nd and Grade 7th students to perform mathematical operations in a visual way. This includes Teachers creating assignment, Students solving them and then Teachers grading them. A web based interactive playground (Google Blockly) can be accessed upon running the application. This playground handles dynamic computation of equations along with providing flexibility to manage history. The operator resrtiction have been imposed based on what grade the student is in.

Here's something about this project

Implementing Machine in Edge Device

This project proposes an architecture to prevent cloud infrastructure from unneccessary traffic. A Machine Learning model is trained in cloud which in turn sends classifier values to the local system. The local system is connected to various sensors using Arduino Uno board. These sensors send real time data to local machine in order to make real time descisions.
If the real time values go beyond the values specified by the classifier, an alarm is triggered by the Fire Detection system. Similarly, a water pump is activated in case of Water Supply System.
